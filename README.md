# Personal Cybersecurity Portfolio

A modern, responsive, single-page portfolio website showcasing my work as a Cybersecurity Engineer and M.Tech Cybersecurity researcher.

The portfolio highlights my professional experience, cybersecurity projects, technical skills, certifications, and contact information through an interactive case-file inspired interface.

## Live Demo

🔗 [https://jatinjayasimha.github.io/Portfolio/]

---

## Features

- Modern cybersecurity-inspired design
- Fully responsive layout
- Dark & Light mode with saved preference
- Interactive project case files
- Project category filtering
- Smooth animations and scroll effects
- Resume preview and download
- Copy-to-clipboard functionality
- Deep-link support for individual projects
- SEO optimized
- Open Graph and Twitter meta tags
- Structured data (JSON-LD) for better search indexing
- Accessible keyboard navigation

## Running Locally

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Navigate into the project

```bash
cd portfolio
```

Open `index.html` in your browser.

No build tools or frameworks are required.
